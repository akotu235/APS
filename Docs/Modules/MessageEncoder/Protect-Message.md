# Protect-Message

## SYNOPSIS
Encrypts the message.

[\\]: # (END SYNOPSIS)

## SYNTAX
```
Protect-Message [-Message] <String> [<CommonParameters>]
```

[\\]: # (END SYNTAX)

## DESCRIPTION
Encrypts the message with the indicated rsa public key and copies it to the clipboard.

[\\]: # (END DESCRIPTION)

## PARAMETERS
