# Confirm-Admin

## SYNOPSIS
Run as administrator.

[\\]: # (END SYNOPSIS)

## SYNTAX
```
Confirm-Admin [[-ScriptBlock] <String>] [-NoExit] [<CommonParameters>]
```

[\\]: # (END SYNTAX)

## DESCRIPTION
Opens a new powershell session as administrator and executes the script block specified in the parameter.

[\\]: # (END DESCRIPTION)

## PARAMETERS
