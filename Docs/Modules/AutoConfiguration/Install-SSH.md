# Install-SSH

## SYNOPSIS
Installs and configures OpenSSH.

## SYNTAX
```
Install-SSH [<CommonParameters>]
```

## DESCRIPTION
Installs OpenSSH.Client.
Installs OpenSSH.Server.
Starts the service.
Set service automatic startup type.
Configures firewall.
