# Use-Speech

## SYNOPSIS
Changes text to speech.

## SYNTAX
```
Use-Speech [-TextToSpeech] <String[]> [<CommonParameters>]
```

## DESCRIPTION
Speaks the text specified in the parameter aloud.
## PARAMETERS

### -TextToSpeech

```yaml
Type: String[]
Required: true
Position: 1
Default value: none
Accept pipeline input: true (ByValue)
Accept wildcard characters: false
```
